Cơ chế bảo mật CORS trong trình duyệt
Do các dịch vụ Agent chạy cục bộ không tích hợp sẵn cấu hình CORS, việc gửi yêu cầu trực tiếp từ file HTML trên trình duyệt sẽ bị chặn. Để demo hoạt động, vui lòng cài đặt tiện ích mở rộng Allow CORS trên Chrome/Firefox hoặc khởi chạy trình duyệt tắt chế độ bảo mật:
chrome.exe --user-data-dir="C:/temp" --disable-web-security
